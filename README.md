# project-euler

## Installation

* [nodejs](https://nodejs.org/)
* [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)

```
npm install
```

## Usage

Run all tests

```
make all
```

Run single test

```
make p001.js
make p001.res
make p001.ml
make p001.rs
```

Watch files and run tests

```
make
```
